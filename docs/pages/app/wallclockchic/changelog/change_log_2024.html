April 2024

5 Fri
Courrie Wall Clock Chic is currently only available for devices with CPUs of the x86-64 architecture.
If the application is launched on a device that does not meet its CPU requirements, it will now exit successfully with a message informing the user of the essentials whenever possible.
A three-minute timer has been added to the application.
