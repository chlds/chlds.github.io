Application / Wall Clock

Courrie Wall Clock is a digital wall clock application for desktops.

The application retrieves the current local time from the computer and displays it on the screen.

You can place the clock in any position* you'd like.

The application is dedicated to being a clock that doesn't interfere with your work.

*If the clock is placed on the taskbar, it may be hidden behind it.


The languages currently supported for date display** are Afrikaans, Albanian, Arabic, Armenian / Hayeren, Azerbaijani / Azeri, Basque, Belarusian, Bosnian, Breton, Bulgarian, Castilian / Spanish, Catalan / Valencian, Central Thai / Siamese / Thai, Chinese, Croatian, Czech, Denish, Dutch, English, Estonian, Farsi / Persian, Filipino, Finnish, French, Galician, Georgian, German, Greek, Gujarati, Hausa, Hebrew, Hindi, Hungarian, Icelandic, Indonesian, Irish, Italian, Japanese, Javanese, Kannada, Kazakh / Qazaq, Konkani, Korean, Kyrgyz, Lao, Latin, Latin in the Roman calendar, Latin Roman numerals, Latvian / Lettish, Lithuanian, Lower Sorbian, Macedonian, Malay, Malayalam, Manx / Manx Gaelic, Maori, Marathi, Nigerian Pidgin, Norwegian, Norwegian Nynorsk, Odia, Pashto, Polish, Portuguese, Punjabi (Gurmukhi), Romanian, Russian, Scottish Gaelic, Serbian, Sindhi, Sinhala, Slovak, Slovene / Slovenian, Swahili, Swedish, Tamil, Telugu, Tongan, Turkish, Turkmen, Ukrainian, Upper Sorbian / Wendish, Urdu, Uzbek, Vietnamese and Welsh.

**It is intended to allow local time retrieved from your computer to be displayed in multiple languages.

About Sticky note functionality

How to enable the sticky note functionality

First, create a folder with the name courrie in the home directory.

Next, create a text file with the name note.txt in that folder.

The above requirements enable the sticky note functionality of the application.

The application allows up to 500 characters*** in that text file to be displayed on the screen.

***The sticky note functionality requires that characters are encoded in UTF-8 format.
